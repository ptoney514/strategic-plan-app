## Key Insights from Data

1. **Status Mapping:**
   - "On Target" = progress >= target
   - "Off Target" = progress < target but close
   - "Critical" = significantly below target

2. **Metric Types Found:**
   - `rating` - 1-5 scale (e.g., 3.74)
   - `ratio` - decimal (e.g., 0.85, 4.1)
   - `score` - percentage-like (e.g., 90)
   - `percent_difference` - vs state average (e.g., +8 = 8% above state)
   - `percent` - percentage value (e.g., 90%)
   - `count` - raw number (e.g., 5,600 office referrals)

3. **Directionality:**
   - Most metrics: higher is better
   - Disproportionality metrics: **lower is better** (closer to 1.0)

4. **Time Series:**
   - Data shows Dec '23 and Current values
   - Annual view available (need to capture if more history exists)
